# Sistema de Gestión de Préstamos de Libros

## Descripción
Este proyecto fue desarrollado en Java y permite gestionar el préstamo de libros utilizando un Árbol Binario de Búsqueda y una Cola Genérica.

## Funcionalidades
- Registrar libros.
- Mostrar libros.
- Registrar solicitudes.
- Mostrar solicitudes.
- Atender solicitudes.
- Generar reportes.

## Tecnologías
- Java
- Eclipse
- Git
- GitHub
